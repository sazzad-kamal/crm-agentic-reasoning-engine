## NEEDS_RAG DECISION

RAG stores unstructured notes/context (not in SQL tables):
- Company notes: key people, relationships, concerns
- Contact notes: preferences, communication style, decision behavior
- Opportunity context: deal notes, risks, blockers, recommendations
- History details: meeting/call summaries beyond dates
- Activity notes: preparation context, what was discussed

needs_rag = true:
- "What's happening with [company]?" -> context needed
- "Why..." questions -> reasoning/concerns
- Questions about notes, concerns, blockers, relationships

needs_rag = false:
- Stage, value, status, dates -> structured SQL fields
- List queries: "all deals", "pipeline deals" -> SQL only
- Metrics: counts, totals -> SQL only
